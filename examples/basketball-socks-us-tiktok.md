# Example: US TikTok Basketball Cushioned Socks (Route A · 整片演绎一次性出片)

> 第二个实战案例。走 **A 整片演绎（单条 Seedance 一次出 15s）**，与宠物梳样例（B 逐镜）互补。
> 这条片在真实生产中跑通并验收过，重点演示了几条把"演绎做不了突变 / 一次性出片必翻车"扭转过来的技巧（见文末「本案验证的关键技巧」）。时间码为**成片抽帧实测值**，非脚本设计值。

## Input

```text
产品：篮球精英减震袜，6 色（白配蓝、白配藏青、米配红、浅灰、藏青配橙、黑配红），
加厚毛圈减震底 + 足弓压缩支撑带 + 精英款剪裁，脚背红色 R logo、脚踝口短标记线。
目标市场：美国 TikTok。人群：篮球 / 运动人群。
```

## 0. Route Decision

- 路线：**A 整片演绎（一次性出片）**。用户明确要一次性出片试，且球场动作（暴扣 / 急停）是"带方向位移"=演绎甜区。
- 画风：**原生直拍流**（户外街球场、金色斜阳、手机直拍质感）。
- **Hero 配色 = 米色配红**（不是黑红）。原因：黑色会"吃掉"加厚毛圈 / chevron / logo 这些要卖的工艺细节，浅色高对比才显工艺；黑红降为结尾六色之一。
- **人物美区本地化**：22 岁美国黑人街球男（默认东亚脸不符市场）。
- 关键取舍：**痛点对比含"换袜"突变**，整片演绎靠插值会把突变摊平 → 用「产品插入格」当切口藏换袜（见技巧 1）；**毛圈底减震硬特写不进演绎**（细密纹理必糊），只做立姿中远景，硬证明留给真图图生后期剪入（见技巧 2）。

## 1. 痒点 & 卖点

- 痒点：穿普通薄棉袜打球，反复起跳落地又震又磨，几局下来脚先于人废。
- 主打利益点：加厚毛圈减震底全程缓冲，落地被"垫"住，打满全场脚不震不磨。
- 一句话定位：为被普通袜子磨脚震脚困扰的篮球人，提供"落地有缓冲"的专业减震袜。

## 2. 15s 结构（5 格 · 真实抽帧时间码）

| 格 | 真实时间 | 作用 | 画面 |
|---|---|---|---|
| 1 | 0–3.3s | 痛点钩子 | 坐地揉只穿薄白袜的脚、龇牙摇头（白袜=旧物，与产品拉开） |
| 2 | 3.3–5.3s | **产品展示=因** | 米色配红袜立姿 + 缓推 + 光带扫，凸显加厚（**不露脚底**） |
| 3 | 5.3–9.5s | 爽点核心=果 | 换上米色袜暴扣慢动作（换袜藏在格2 切口外） |
| 4 | 9.5–13.3s | 爽点延续 | 落地缓冲 + 急停变向、自信 |
| 5 | 13.3–15s | 收口 | 六色一字排开快闪 + CTA |

产品 showcase 放**正中间**当因果枢纽 → 观众潜意识把"变强"归功于这双袜（问题→产品→结果）。

```text

时间分配：格1 痛点约 4.5s（连续演满：揉脚、龇牙、摇头、烦躁，不静止）、格2 产品展示约 3s、格3 暴扣慢动作约 4s（最重要、从容演满不许加速）、格4 落地缓冲+急停约 2.5s、格5 六色快闪约 1s。
格间衔接：格2 是产品镜头（场景切换、球员不出现），换袜藏在这格切口里；格3 球员已穿米色配红袜出场，不在身体上表现换袜、不保留白袜；格2 产品只允许镜头极缓推近 + 光带扫过袜身。
```

## 5. 剪辑执行表（真实抽帧时间码 · 单条成片）

单条整片，无拼接 / 无转场。第一步先关 AI 片段原声。

### 5.1 BGM / 切点
- BGM **beat drop 对齐 `5.3s`**（产品切到球员爆发的翻转点）；前低沉、后 hype。
- 单条片无段间切点，靠 BGM 自动踩点把情绪卡在画面节拍上。

### 5.2 音效 cue（真实时间）
| 时间 | 音效 | 画面 |
|---|---|---|
| 2.5s | 叹气 / 泄气 | 痛点揉脚 |
| 3.3s | whoosh | 痛点切产品 |
| 5.3s | whoosh + beat drop | 产品切球员 / 起跳 |
| 7.0s | 篮球入网"刷" + 轻呼 | 暴扣进筐 |
| 12.5s | 球鞋急停"吱" | 落地急停 |
| 13.3s | pop / 亮相 | 六色出现 |

### 5.3 字幕逐条（真实时间）
| # | 时间 | 字幕 | 位置 |
|---|---|---|---|
| 1 | 0–3.3s | Cheap socks quit before you do. | 距顶 25%（脚在下半屏） |
| 2 | 3.3–5.3s | Thick terry-cushion sole. | 距底 ≥15% |
| 3 | 5.3–9.5s | Land soft. Hoop harder. | 距底 ≥15% |

| 5 | 13.3–15s | 6 colorways — tap to cop 🏀 | 距底 ≥15%（红底条） |

字体无衬线粗体、白字 + 2–3px 黑描边（白底产品帧改黑字）；高亮词用产品红。**底部字幕距底 ≥15%**，否则被 TikTok 底部 UI 盖住（实测成片初版偏低）。

### 5.4 导出 + 封面
- 9:16，**1080×1920**（剪映默认常 720P，手动改），≥30fps，H.264 mp4，标注 AI 生成。
- 封面：~6.5–7.0s 暴扣腾空帧 + 大字 `BALL ALL DAY`，避开乱码小字帧。
- 终检（ffprobe/抽帧）：分辨率 1080、音频无削波（峰值 ≈ −0.5dB / 均值 ≈ −19.7dB 健康）、字幕在安全区。

## 6. 合规提醒（美区）
- 不宣称医疗 / 绝对化（"防崴""治脚痛""100%"），改用"缓冲 / 更稳 / 更舒适"。
- 图实一致：毛圈厚度、减震效果别夸大（实物要真有加厚毛圈底）。
- 按 TikTok 要求标注 AI 内容；TikTok Shop 资质以当地最新政策为准。

## 7. 本案验证的关键技巧（链接到主流程）

1. **「产品插入格」破解换装突变**：痛点白袜 → 产品特写格（场景切换、人不在画面）→ 爽点已穿新袜，换袜藏在切口外，单条演绎不 morph。见 `references/seedance2-multimodal.md` 硬边界破解。
2. **一次性演绎避开脚底/细密纹理硬特写**：演绎只放立姿中远景，毛圈底减震硬证明单独真图图生再剪入（产品分级出镜在 A 路线的落地）。见 SKILL 3.1 / seedance2。
3. **深色吃工艺，hero 用浅色高对比**：黑款生成出来加厚/纹理全糊，改米色后 chevron 与 R 清晰。见 SKILL 3.5 产品形态约束。
4. **产品展示放正中间 = 强化"因为它才变强"的因果**。见 SKILL 第 2 步脚本结构。
5. **成片优先定卡点**：剪辑时间码用 ffmpeg 抽帧读真实节奏，不按脚本设计值（演绎节奏由模型定，必偏移）。见 SKILL 后期剪辑。
6. **提示词全程对模型说话**：删元话术、正向描述别点名不想要的颜色；颜色按实物不擅自改色。见 SKILL 3.5。
