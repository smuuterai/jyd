# UGC 证言 / 口碑测评带货（Testimonial）

带货脚本结构的一种变体（对应主文件文末"测评款"）。当产品靠**真实用户证言 / 口碑 / 前后体验 / 痛点见证**转化时用它。命门：**证言要"看着像真用户随手发"，不是"广告里的好评"**——一旦像摆拍念稿，可信度归零、转化崩。

> ⚠️ **真实性红线（本技能立场，外部教程不提）**：AI 生成的"证言 / 好评 / 数据"若是**编造的虚假反馈**＝虚假宣传，平台限流封号 + 法律风险。正确用法只有两条：①把**真实用户的真实反馈**做可视化呈现；②或明确标注"演绎 / 示意"。**绝不编造虚假好评、虚假数据、虚假身份**。下面所有技巧服务于"把真实证言拍得可信"，而不是"把假证言拍得像真"。

## 4 个 2 秒钩子（证言专用）
- **结果先行**：开场怼数字 / 成果，再揭露背后的人（"3 周瘦 8 斤"→ 人出场）。量化结果强时用。
- **情绪抓拍**：先给真实反应瞬间（惊喜 / 释然），再补上下文。有真人情绪素材时用。
- **数字冲击**：大数字填满画面、count-up 弹入。ROI 清晰的对比型用。
- **前后对比**：闪现 before 立刻接 after。视觉差异明显的转变型用（与 §3 前后对比互通）。

## 4 种视觉格式
- **纪录片访谈**：人物坐三分线偏侧、暖光访谈布光（主光 45°、补光暗 1.5 档）、环境强化身份、固定或微漂移机位、2 秒处打下三分之一身份字幕。
- **数据可视**：深色底、图表从基线动画到结果、归属字幕、全程缓推。
- **分屏旅程**：画面横向分割，上＝去饱和 before / 下＝高饱和 after，约 3/4 处分割线让 after 占领画面。
- **快剪堆证**：3–7 个 1–2 秒短镜交替"脸 + 数字"、统一布光、卡 BPM、收在汇总数据。

## 8 条带货证言铁律
1. **结果先于人**：前 2 秒先给证据再给人——观众先决定信不信。
2. **具体数字 > 泛泛承诺**（⭐通用于一切带货文案）："营收翻倍"弱，"从 12K 做到 31K、用了 11 周"强。要具体数字 + 时间 + 可命名结果。
3. **人声为主**：证言人声是第一音频信号（-6dB、居中、高通 80Hz、3–5kHz 提亮），BGM 压 -18~-20dB 垫底；标准＝每个字不费力就听清。
4. **五星必带平台来源**：光给五星不标来源＝像编的；哪怕很小也要带平台名（Amazon / TikTok / 当地平台）。
5. **匹配受众气质**：B2B 要权威（干净环境 / 专业布光 / 数据）；C 端要共鸣（真实空间 / 自然光 / 生活瞬间）。
6. **每句证言要挣得屏幕时间**：超 3 秒的引言要配画面运动 / 补充画面，别干放字。
7. **收尾＝承诺锚点**：最后一帧给"产品在场景里 / 用户带结果归属 / 聚合社会证明"，别泛泛收。
8. **避免"代言姿势"**：别对镜大笑念稿——**看镜外、真实表情、具体语言**远胜摆拍式好评。这是"防假证言"的核心。

## 防假证言（真人三限定的证言版）
- 看镜外、不对镜傻笑；表情随话语真实流动；用具体细节代替"超好用"。
- 真人皮肤 / 眼神 / 重量感按 [[subject-and-realism]] 的真人三限定写。
- 环境带生活痕迹（真实房间 / 工位），别棚拍式干净（除非走 B2B 权威定位，见画风轨道·棚拍广告流）。

## 进阶微 craft（潜意识强化，可选）
- **引言处 1% 微推**：客户说关键引言时，镜头在约 3 秒内推近 1%——意识察觉不到、潜意识感到"情绪在加强"。
- **Success-glow 打光**（转变 / 成果型证言）：暖金主光（2700–3000K）从眼上方 30°（非 45°）像晨昏光打脸，比正常访谈过曝约 1/3 档，营造"人生高光"感。配 [[luxury-aesthetic]] 暖调控制。

## 本地化
证言人物用目标市场本地外貌、说当地语言；平台来源本地化（美区 Amazon / TikTok review，巴西用当地平台）；数字、货币、单位本地化。
