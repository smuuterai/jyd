# 高端 / 奢品 / 美妆带货的"克制美学"

棚拍广告流（见画风轨道）的高端细化版。香水 / 护肤 / 奢品 / 珠宝 / 高端家居等**靠"质感与氛围"而非"参数"转化**的品类用它。核心＝**克制 > 堆砌**：每个元素都要挣得它的位置，一物一空间，杂乱＝不安全感＝掉价。

## 十条克制铁律
1. **大留白**：负空间占画面 40–60%——空＝贵的信号。
2. **慢**：运镜 0.1–0.3x 速度（缓推 5 秒只走 10–20% 画幅；缓摇 ≤15°/5 秒）。禁手持抖、快摇、急推、荷兰角、俯冲。
3. **明确写"画面里没有什么"**（"无杂物、无多余道具、无文字"）——奢侈感来自减法。
4. **材质微距＝产品本身**：怼拍纹理 / 编织 / 纹路 / 瓶身折光（material honesty），材质就是卖点，别只写"漂亮瓶子"。
5. **动作后留 settling time**：运镜停下后让画面"沉"一拍，比一直动高级。
6. **静默开场 2–3 秒**再进环境音（"silence as design"）。
7. **去饱和 / 单色控制**：60–70% 饱和或单一色系（全冷或全暖）；黑金＝深黑 #0a0a0a + 选择性金；奶油＝暖奶白高光 + 深暖黑。
8. **每个光源给 Kelvin**：柔雕塑光 / 单硬光戏剧 / 磨砂日光 5500K / 烛光 1800–2200K。
9. **声音**：单音钢琴（小调 40–80BPM、音符间留白）或慢弓大提琴；禁打击乐 / 亮合成器 / 人声钩子。
10. **写可测量的具体**，别用"美 / 惊艳"等空词；先定环境再定主体。
