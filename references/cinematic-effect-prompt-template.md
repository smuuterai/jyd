# 电影级特效片·提示词模板（奇观/特效镜，Veo 3 / Sora 级）

> **何时用**：要做"电影级写实特效"短片，或给带货做奇观钩子——超速奔跑、时间静止、力量爆发、物体炸开、刚体破坏等。
> **用哪个模型**：靠 **Veo 3 / Sora / 可灵**这种能扛复杂物理的模型；**即梦 Seedance 做"时间静止+悬浮碎片"这类会吃力**（它甜区是连续动作/整片演绎，不是复杂特效物理）。普通带货片别强上特效，确有 Veo3/Sora 入口再用。
> **来源**：拆自一条写实"超速+时间静止"特效片（主角纽约街头疾跑→世界冻结、行人汽车悬停、报纸掀飞、混凝土碎裂悬浮→时间恢复、碎片砸落，15s 横屏，执行优秀）。

## 第 0 步：先过"能不能生成"判据（别被漂亮玄幻稿骗）

照搬 `video-deconstruction.md` 的死穴判据，出特效提示词前先自检：
- ✅ **能做好**：带方向的高速位移（奔跑/冲浪/跳跃/腾空转体）；环境**完全冻结**（时间静止——冻结＝静态，反而好画，只让一个主体定向移动）；刚体破坏的**结果态**（碎片已裂开/悬浮/坠落）；光带扫过、物体铺开/生长。
- ⛔ **死穴（避开或改）**：**原地自转/旋转**（转圈、产品 360°、旋裙）；**满屏高速粒子/神兽特效**；**唱跳精准对嘴**。同时出现"原地旋转 + 复杂粒子 + 多主体对嘴"两项以上 ≈ 必崩纸面稿。
- **改法**：把"原地高速旋转"改成"带位移的转 + 低位跟拍"；把"满屏特效"改成"单一可控特效 + 主体小幅动作 + 背景特效层"。
- **一句话**：炫的部分必须落在"**定向运动 + 冻结环境 + 结果态破坏**"，不是"原地自转 + 满屏粒子"。这条片所有炫点都避开了死穴，才稳定出片。

## 三个写法要点（这条片为什么稳）

1. **逐秒分"机位 / 物理 / 音频"三层**：控制力远超一段散文。物理那层要写"**怎么发生**"（音爆扭曲空气、鞋陷进混凝土、碎片先悬浮再坠落），不只写"很快"——写过程不只写状态。
2. **负面提示专门压"游戏感 / 发光 / CGI"**：逼出"接地气写实"。同带货"原生直拍流"逻辑——越像真实素材越可信，一发光就廉价。
3. **主角锚 + "严格遵守原版服装、全程稳定面部"**：跨秒不变脸、不换衣；高速运动里尤其要这句压住漂移。

## 带货怎么用

当 **0–3 秒奇观钩子**：力量 / 速度 / 科技调性的产品（球鞋、球袜、运动、数码），用"超速 + 时间静止"开场拦截滑动，再 bait 到产品镜。这是 `video-deconstruction.md`「奇观化」梗在特效层的极致版。⚠️ 高门槛生成，只在确有 Veo3/Sora 入口时用；要稳、要便宜就回常规带货结构。
