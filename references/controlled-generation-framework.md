# 受控生成总纲（产品无关 · 举一反三）

> 从工业设备类 + 萌宠剧情类等多个实战项目抽象出的根因框架。出任何商品的视频提示词前，按本文件过一遍，预防 90% 翻车。

## 一条总根因（注意：锅在写提示词的人，不在模型）
翻车的真正根因**不是"模型会乱填"，而是"我在猜模型会怎么做、而猜测总是错的"**。每次心里冒出"它应该会自动补上 / 它大概会硬切 / 它会读图不读字 / 这样它应该懂"——**这都是赌博**，赌输就翻车。
**纪律：不预测模型，只规定模型。** 凡是你心里出现"它大概会…"的念头，那就是一个**必须立刻写成显式指令**的漏洞——把这个猜测在提示词里写死，猜测就不存在了，也就没有赌输的可能。下面六个维度是"最常被拿去赌"的高发区，逐一写死、不留任何"它应该懂"的余地。

## 六个必锁维度（任何商品通用）

### 1. 身份 / 外形（东西长什么样）
- **怎么失败**：产品/主角/道具在运动中变形、logo 乱码、换脸、道具和产品混淆。
- **通用规则**：真图垫底锁外形；每条提示词**完整重述**外形不省略；硬细节品类"光动机不动"（只一个运动元素、锁机位锁机身、禁旋转>20°）；惨状/对比道具与主推产品在颜色形态上拉开。
- **跨品类**：手表/数码=锁表盘按键禁旋转；折叠屏/翻盖瓶=锁开合方向+内容物颜色质地（点名禁止项）；服装=面料剪裁词锁质感；带字包装=刺绣/logo 禁糊，含字镜头<2.5s 不定格、不做封面。

### 2. 空间（在哪、几何、方向、光）
- **怎么失败**：同一场景每镜几何不同（沙发高度/地板材质变）、左右方向打架、主体凭空出现在已建立为空的区域。
- **通用规则**：开拍前定**空间地图**（谁在左谁在右、光从哪来、关键物在哪）；同地点第二次出现用第一次的帧当锚（锚的是地点不是把全片锚到一帧）；**"此处为空"也是事实**不可违背；多角色给**家位分离**（睡点/基地/事发点分开，距离=奔赴戏）。
- **跨品类**：厨房用品=灶台水槽固定位；任何"使用前后对比"=同机位同地点否则对比不成立；丢失/寻回类=丢的地方必须=找回的地方。

### 3. 时间（每拍多久）
- **怎么失败**：剧情多的段被模型加速跑过、关键过程被压缩跳过。
- **通用规则**：按总时长把秒数分到每段，**动作密集段明显多给**，写明"这段 X 秒演满不许加速"。先算账再写提示词。
- **跨品类**：搓泥/起泡/吸黑头等"过程爽点"=核心爽点段给足时长；开箱=开盖动作别赶。

### 4. 状态突变（变身/出现/消失/前后对比这类不连续变化）
- **怎么失败**：连续生成的插值会把突变**摊平成渐变**（披风提前出现、效果从第一帧就糊在脸上）。
- **通用规则**：突变**不能靠文字"硬切"在连续生成里完成**。两条解：①**藏**——把突变藏进遮挡物里，给模型一个"空窗"瞬间；②**切**——逐镜生成两段+剪映黑场/硬切拼接。累积型效果（泡沫/泥条/黑头）一律用**结果静图**呈现，不靠连续生成累积。
- **"藏突变"的遮挡物工具箱**：实体遮挡（钻进屋子/盒子/被子——场景里要正好有）｜**烟雾/光爆**（一团复古白烟在画面中央炸开，烟散去突变已完成——**最通用，随处可炸不依赖场景**，配 "Poof" 音效）｜转身背对镜头｜物体划过挡镜头｜镜头被遮黑。变身/换装/出现消失类，烟雾遮挡是首选万能解。
- **跨品类**：化妆品涂抹前后=藏在"手抹过脸"的遮挡或拆两段；开箱"封闭→打开"=盖子挡镜头瞬间完成；任何 before/after。

### 5. 过程 / 动作（A 怎么变成 B、状态间的位移）
- **怎么失败**：只写每段的**到达状态**、不写"怎么到达"，模型跳过过程直接给结果（老鼠像凭空变出来的）。
- **通用规则**：每段除"到达后状态"，必写"**如何从上一段到达本段**"的衔接动作，逐拍展开。要展示的过程逐拍写；要隐藏的过程（见维度4）才藏。
- **跨品类**：产品"从桌上被拿起→开盖→挤出→涂抹"每一拍都要写，不能写"涂着产品的脸"就完事。

### 6. 镜头（机位 / 景别 / 运镜）
- **怎么失败**：不写机位，构图随机；想要的推近/俯拍/特写得不到。
- **通用规则**：每段写死**景别+角度+运镜**。演绎路线机位主要靠故事板格子的构图承载，逐镜路线靠文字。
- **⭐ 红线轨迹规划法（运镜的图形化控制，实测有效）**：在参考图上**画一条红线标出镜头运动路径**，提示词写"相机严格按红线顺序运动"，**并务必加一句"红线仅为路径草图，最终视频绝不能出现任何红线、箭头或标记"**——模型会照红线走位但不把线渲染进画面。比纯文字描述运镜更直观可控。（修正旧结论：箭头/标记**可以**画在参考图上指挥运镜，关键是明令成片不出现标记；之前"带标记图不能用"的说法只适用于"标记没被声明排除"的情况。）
- **一镜到底巡游结构（多主体/多SKU展示）**：相机贴地低机位连续飞过多个对象，每个对象前停顿+动作，最后拉全景收尾。多人介绍、产品阵列、SKU 展示都可用——比逐个硬切高级、沉浸感强，且落在 Seedance 甜区（连续平滑运镜，无高速旋转）。配红线规划走位。
- **多主体可分配不同语言口播**：一条里指定"某人用德语、其余中文"等，模型能按人物分配语种——多市场/多语种带货可用。

**运镜速查表（想要什么情绪→用哪个运镜→怎么写）**：
| 运镜 | 效果/用途 | 提示词写法 | 带货用在哪 |
|---|---|---|---|
| 推近 Push-in/Dolly-in | 制造亲密/紧张 | "缓慢推近，结束于特写" | 产品登场、卖点强调 |
| 拉远 Pull-back | 揭示场景/尺度/收尾 | "缓慢后拉揭示全景" | 开箱后展示全貌、结尾 |
| 摇 Pan | 横向揭示空间 | "缓慢左摇到右" | 扫过产品阵列 |
| 升降 Tilt | 纵向展示高度 | "缓慢上摇" | 展示产品高度/细节从下到上 |
| 跟拍 Tracking | 跟随运动、沉浸感 | "侧向跟随主体" | 运动类、使用过程 |
| 环绕 Arc/Orbit | 多面展示、英雄时刻 | "缓慢环绕，≤30度" ⚠️超30度产品会崩 | 产品展示（小幅）|
| 吊臂 Crane | 宏大上帝视角 | "缓慢吊臂上升到俯瞰" | 场景大、收尾 |
| 变焦 Zoom | 聚焦/隔离，比推近平 | "缓慢变焦推近" | 突出单点 |
| 推拉变焦 Dolly Zoom | 眩晕/心理冲击 | "前移同时变焦拉远" | 戏剧高潮（⚠️复杂运镜，即梦能否稳定做未验，慎用）|
| 快速摇 Whip Pan | 转场、停滚动 | "快速右摇接下一镜" | 镜头间转场 |
| 手持 Handheld | 真实/原生感 | "手持，轻微抖动" | UGC 原生流必备 |
| 静态+角度 | 低角=力量/俯=尺度/荷兰角=不安 | "低角度仰拍，固定" | 产品显气场用低角 |

**⭐ 运镜两大铁律（外部金句，与实测完全吻合）**：
1. **一个运镜一个片段，永远如此**——别在一条提示词里堆多个运镜（="一镜一动作"在运镜层的版本，堆了必乱）。
2. **几乎所有运镜都加"缓慢"**——慢动作藏住 AI 渲染不清的部分，快速移动暴露每一个缺陷。这是"运镜写小写慢/光动机不动"有效的根本原因：**慢藏丑、快露馅**。要快速运镜（whip pan）只用在转场一闪，不用在要看清主体/产品的镜头。

### 7. 动作密度（每镜只给一个核心变化）
- **怎么失败**：一镜塞多个变化/多拍动作，模型演一两拍就发呆或糊（铁律"持续性"）。
- **通用规则**：每镜只安排**一个**核心动作或变化（这镜只"猫耳展开"、下镜只"穿上制服"），多了拆镜。逐镜时间码写法的样板：`时间码｜景别+运镜｜一个核心变化｜光线质感` 一行流。
- ⚠️ **"逐渐形变/累积型变化"是死穴**（皮肤蒸发露骨架、泡沫渐起、毛发渐多）——别想在连续视频里做完整过程，用结果静图或遮挡（维度4/5）。复杂变形+高速粒子+glitch 同屏=必崩纸面稿（见 video-deconstruction 快速判据）。

## 两条路线 = 两个相反的控制面

| | 逐镜路线 | 演绎路线（故事板→连续生成）|
|---|---|---|
| 控制靠 | **文字**（每镜写到极细）| **故事板图**（视觉/空间/外形细节画进格子），文字只管骨架 |
| 拼接 | 剪映拼 | 一次连续生成 |
| 成本 | 慢、贵 | 快、便宜 |
| 用于 | 精确卡点/运镜/慢镜头、有硬突变、产品英雄特写要保真 | 连续动作剧情、萌宠日常、走秀滑冰、无突变或突变可藏 |

**演绎路线文字能管什么 / 不能管什么**（实测）：
- 能管：时间分配、衔接过程动作、高层状态、藏突变（"完全消失进遮挡物"）、顺序、一致性锚、负面词。
- 不能管：在连续生成里硬切（自相矛盾，切是剪辑操作）、阻止未隐藏突变被插值摊平、精细产品细节（归图）、精确运镜数值。

## 任何新商品 · 开拍前自检（跑一遍预防 90% 翻车）
1. 产品哪些细节是"命门"（logo/形状/开合/颜色）？怎么锚？
2. 有几个地点？画一张空间地图（左右/光向/关键物位置）。
3. 有没有状态突变（变身/前后/开合/出现消失）？→ 决定 **藏** 还是 **拆段切**。
4. 有没有"累积型效果爽点"？→ 用结果静图，不靠连续生成。
5. 每段之间的位移/衔接动作写了吗？
6. 总时长怎么分配？动作密集段够秒数吗？
7. 每段机位写了吗？
8. 选哪条路线（逐镜/演绎）？
9. 负面词：禁文字乱码、禁多主体、禁变形、禁错误地板/背景。
10. 任何修订 → 重发全量排序版，不让用户跨消息拼版本。
