# 翻车速查表（开拍前扫一遍 · 一页索引）

> 这是绝活的总目录。每条 = 一个实测翻车现象 → 一句话对策 → 去哪条细看。
> 出任何分镜前过一遍；命中哪条就翻到 skill.md 对应小节。根因总纲见 `controlled-generation-framework.md`。

## A. 产品 / 主体保真

| 翻车现象 | 一句话对策 | 细看 |
|---|---|---|
| 运动中产品变形、logo 乱码、换脸 | 真图垫底锁首帧 + 每条**完整重述外形** | 铁律 3 / 6、3.4 |
| 硬细节产品转 20°+ 就重新设计、画出没见过的面 | **光动机不动**：只一个运动元素、锁机位锁机身、禁转>20° | 铁律 6 |
| 纯俯视(80–90°)逼 AI 编"机顶/窝顶"、logo 必崩 | 降到 **45° 高位斜俯**，仍能看见正面锚 | 铁律 6 |
| 文生视频里产品特写必穿帮 | 产品特写**只**走"真图+图生视频"；多场景镜让产品只中远景 | 3.1 |
| 上传真人照 → AI 出"原图美化版" | 身份参考三步：保留人不保留原图 + 身份锚拆清单 + 发型三变量 | 3.4 |
| 为突出卖点擅自给产品改对比色（黑袜底改米白突出减震） | 颜色严格按实物，卖点靠纹理/动作/光带/特写体现，不靠改色 | 3.5 产品形态约束 |
| 黑/深色款生成出来工艺细节全糊（加厚/纹理/logo 看不见） | hero 选浅色高对比款展示细节，深色款留作结尾系列展示 | 3.5 产品形态约束 |
| 一次性整片演绎里塞产品细节硬特写（翻底/微距/小字）必糊 | 演绎只放立姿/中远景，细节硬特写单独真图垫底图生再剪入 | seedance2 / 3.1 |

## B. 效果 / 突变（AI 视频死穴）

| 翻车现象 | 一句话对策 | 细看 |
|---|---|---|
| "梳出一堆毛/倒满泡沫/吸出黑头"开头就糊上去 | 删**触发词**（凌乱/大量/堆积…）换中性运动词 | 铁律 4 |
| 累积型爽点连续生成必糊 | 用**静图呈现结果**：A 首尾帧 / B 结果图+图生 / C 蒙太奇硬切 | 铁律 5 |
| 变身/换装/前后对比被插值**摊平成渐变** | **藏**（遮挡/烟雾爆开）或**拆段切**（黑场/硬切） | 框架维度4 |
| 带货前后对比想单条整片演绎、又怕换装翻车 | 中间插一格**产品特写**当切口（真场景切换），后格主体已是新状态出场、不演换装（实测成立） | seedance2 硬边界破解 |
| "使用前惨状"在连续视频里造出来很假 | 做成**一张 AI 静图**再轻微运动 | 铁律 5 |

## C. 空间 / 动线（剧情类高发）

| 翻车现象 | 一句话对策 | 细看 |
|---|---|---|
| 同场景每镜几何变（沙发高度/地板材质/光向乱） | 开拍前定**空间地图**，左右光向物位写死 | 3.3 规矩1 |
| 同地点二次出现又被 AI 重新发明 | 用第一次过关的帧当锚再合成 | 3.3 规矩2 |
| 主体凭空出现在已建立为"空"的区域 | **"此处为空"也是事实**，锚的是地点不是同一帧 | 3.3 规矩1 |
| 变身后瞬移到沙发边、观感突兀 | 叙事节拍"信号→反应→行动→结果"，每次位移给移动镜 | 3.3 规矩3 |
| 多镜朝向打架（奔跑向左、营救朝右） | **动线审计表**逐镜核朝向；无字无产品的格水平镜像改向 | 3.3 规矩4 |
| 剧情类用文字概要表对齐 → 必翻车 | 骨架确认环节直接出**可视化故事板**（标注版） | 3.3 规矩5、第0步 |

## D. 动作 / 运镜

| 翻车现象 | 一句话对策 | 细看 |
|---|---|---|
| 大动作（转头/抬手/起身）被缩水到 10–20% | 终点画面描述幅度 + 度数过量 + 大动作提前 + 首尾帧锁姿态 | 3.5①幅度 |
| 一镜塞 3–4 拍 → 演一两拍就静止发呆 | **一镜一拍**，多拍拆多段尾帧承接 | 3.5①持续性 |
| 运镜幅度大时凭空多出第二只动物/人 | 运镜写小写慢 + 正向写"画面始终只有一只 X" | 3.5、框架维度6 |
| 堆多个运镜 → 必乱 | 一个运镜一个片段；几乎所有运镜加"缓慢"（慢藏丑快露馅） | 框架运镜铁律 |
| 原地自转/满屏高速粒子/唱跳对嘴 | 死穴，别做；带方向位移(奔跑/冲浪)反而能做 | video-deconstruction 快速判据 |

## E. 文字 / 真实感 / 转场

| 翻车现象 | 一句话对策 | 细看 |
|---|---|---|
| AI 生成的小字(面单/标签/屏幕)必乱码 | 一律后期加字；含字镜头<2.5s 不定格；封面绝不用 | 铁律 6 补充、3.5 |
| 真人像"AI 假人"塑料脸 | 真人三限定：皮肤毛孔 + 重量感 + 眼神光 | 3.5①真人三限定 |
| 情绪镜头演员平静像好员工、钩子全废 | 情绪词前置 + 具体动作动词落地；验收重抽 | 3.5①情绪、3.8 |
| 依赖生成内容的转场没生成出来（纸只吐半张） | **每个转场必带后备切法**（白闪/硬切），别为转场重抽 | 3.3 |
| 场景再美但与主体身份矛盾（家宠跑荒野） | 先过"合理性"再谈保真：这主体真会在这环境吗 | 3.3 空间保真 |
| 穿戴品想露本体→主体只穿产品没配套装备（袜子没鞋打球） | 配套装备逐件写全 + "避免"点名赤脚/缺鞋；用脱鞋揉脚等动作自然露产品 | 3.5① |
| 痛点镜"疼痛/受伤/冲击"措辞被海外模型(Veo/Sora)审核拦截 | 改情绪向措辞(累/烦/不适/嫌弃)，去掉冲击受伤词；即梦一般不卡 | 3.6 |

## F. 流程纪律（最容易省掉的）

| 翻车现象 | 一句话对策 | 细看 |
|---|---|---|
| 接到"出图/出提示词"闷头就出全量 | **先确认形态/数量/用途/画幅 → 给骨架点头 → 才出全量** | 第0步铁规 |
| 改了一两镜只发改动的 | **任何修订重发完整排序全套**，不让用户跨消息拼版本 | 3.5③ |
| 提示词混入对用户说的元话术（擅自/关键/不许） | 全程对模型说画面事实；正向描述别点名"不想要的"颜色（图片生成会被诱导） | 3.5 开头 |
| 转场默认硬切 | 本用户偏好**无缝**（尾帧承接/遮挡/白闪），硬切仅快剪确需 | 3.3 切点、后期 |
| 全片每段各自色温，切一次跳一次="生硬" | 一个滤镜+一个调节层覆盖全片统一色调；切在动作中不切静止帧 | 后期·三大根因 |
| 按提示词设计时间码出剪辑、卡点对不上真实节奏 | **成片优先**：抽帧读真实时间码再定卡点；提示词版须标注"待成片重对" | 后期剪辑 |
