# 图生视频：把一张静图"动起来"的方法（image-driven）

§3.4 准备好首帧/参考图后，这份管"怎么让它动得真、不崩、过审"。核心：**图已经给了约 80% 信息，文字只补缺口、别复述**（实测：271 字过审、1874 字被拒——短 > 长，同 seedance2「控制在图不在文字」）。

## 先判两种模式（动手前先问用户要哪个）
- **沉思模式（日常 / 偷拍感，最稳）**：主体自然存在、无情绪弧。3–4 个持续微动作 + 静态环境 + 镜头最多缓推 15%、零剪零摇。适合生活感"被偷拍的一刻"。
- **反应模式（高戏剧 / 情绪转变）**：主体有情绪变化（惊讶 / 害羞 / 醒悟）。按**情绪子拍**展开，受物理时长下限约束（见下）。

## 反应时长下限（物理常数，压不下去；拆子拍写，别一句带过）
脸红 ≥2.5s（视线飘 0.8s→低头 0.5s→脸颊红晕扩散 1.2s）｜哭 ≥3s（眼湿→颤→泪落）｜惊吓 ≥1.2s（瞳孔缩→身体紧→表情变）｜笑 ≥2s（眼角→嘴角→带动肩）。别写"她脸红了"，写成上面的子拍序列。

## 三层动作分级（防"雕像效应 / 穿帮"，替代泛泛"自然动作"）
1. **持续微动**（呼吸、肩部浮动、眨眼、发丝飘）——全程不停。
2. **聚焦节拍响应**（嘴角轻扬、轻点头、视线锁 ~0.5s）——定时、克制。
3. **禁止大动作**（站起 / 走 / 转身 / 操作物体）——**显式列出"不要生成 X"清单**。
> 每个动作用"正向夸张 + 负向否定"双锁（"嘴唇明显在唱（不是轻微哼）"）。多人镜头：写"跨主体互动事件"当叙事骨架 + **锁静态全画面**；别写"镜头摇到第 X 人"（触发换脸重生成）；每个位置（左 / 右后 / 前景 / 远处）各写明动作，不止"没人静止"。

## 文字只补 4 类（图已含的别复述）
①图里**没有**的上下文（画外信息）②图里**有但要强化**的细节 ③**避险 / 避审** ④**硬合规约束**（画幅 / 时长 / 无脸 / 禁词）。其余交给模型对图的视觉推断。

## 极简过审结构（女性 / 室内向尤其按这个）
```
[时长][一句前提][画幅]。
参考@图片1[用途说明]，[不要复刻什么]。[整体演变方向]，转场交模型自由设计。
[一句运镜]。[一句氛围 / 色 / BGM]。[关键时刻或分段定位]。
[硬约束：无文字 / 水印 / 可识别人脸 / 合规风险]。
```
> 合规雷区（女性 + 室内触发模型拒绝的措辞）见主文件 §5「措辞合规」。字数硬上限 **2000**（用 `len()` 量、非估），目标 ≤1900 留 100 字余量。

## 夜景 / 手机暗光真实感（防极简提示词把夜拍渲染成影棚，实测）
极简提示词省了降质层 → 夜景 / 手机内容会被渲染成影棚摆拍。夜拍必加三件：
1. **低光环境光**：主体只被环境侧光 / 背光微照、大部分阴影、高对比、无补光、主体别比环境亮、脸不过曝。
2. **手机暗光降质**：高 ISO 噪点 + 阴影 banding + 自动白平衡漂移 + 手持模糊 + 对焦呼吸 + 广角暗角；禁 4K 锐利 / 电影级稳定 / 影棚灯 / 精准调色 / 刀锐对焦。
3. **自然姿态**：随手一瞥代替"对镜转头"、柔和笑代替灿笑、手与头发随走路自然动，去掉夸张摆拍。
